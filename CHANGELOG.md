# سجل التغييرات — دربك (عدّاد الإشارات)

خريطة أقسام سريعة لهذا المستودع، لتوفير الوقت والتوكنز في أي جلسة Claude قادمة:

```
index.html        → الملف الرئيسي (HTML + CSS + JS مضمّن بالكامل، بلا اعتمادات خارجية عدا خطوط Google Fonts)
manifest.json      → إعدادات PWA (اسم التطبيق، الأيقونات، الألوان)
icon-16.png         → أيقونة favicon صغيرة
icon-32.png         → أيقونة favicon
icon-180.png        → أيقونة Apple touch (iOS)
icon-192.png        → أيقونة PWA (تُستخدم أيضًا في نافذة "حول")
icon-512.png        → أيقونة PWA عالية الدقة
CHANGELOG.md        → هذا الملف
```

كل الملفات في المجلد الرئيسي مباشرة (بلا مجلدات فرعية) لسهولة الرفع على GitHub Pages.

## بنية `index.html` (للمرجعية السريعة)
- `<style>` : تصميم الواجهة (بطاقات، إشارات، أزرار، نافذة "حول")
- `.app` : الحاوية الرئيسية — الهيدر، اختيار الاتجاه، شريط الإشارات العشر، بطاقة الإشارة الحالية، لوحة الإحصائيات
- `.about-overlay` : نافذة "حول التطبيق" المنبثقة (معلومات المطوّر وحقوق النشر)
- `<script>` : منطق التطبيق (تخزين محلي عبر `window.storage`، حساب المتوسطات، مؤقّت الإشارة الحمراء/الخضراء)

- **اتجاه الصفحة الديناميكي**: تتحوّل الصفحة تلقائيًا بين RTL (عربي) وLTR (إنجليزي/تركي) عبر خصائص CSS منطقية (`inset-inline-start`, `text-align: start/end`) بدل الخصائص الفيزيائية الثابتة، بحيث تنعكس الواجهة بشكل صحيح.
- **سهم الاتجاه الديناميكي**: سهم "الإشارة التالية" واتجاه أزرار التنقّل بين المدينتين يتغيّر تلقائيًا (← بالعربية، → بالإنجليزية/التركية) ليطابق اتجاه القراءة.
- **الأرقام**: تُعرض بالأرقام الهندية العربية (١٢٣) في وضع العربية، وبالأرقام اللاتينية (123) في وضعي الإنجليزية والتركية.
- **حفظ تفضيل اللغة**: يُحفظ اختيار اللغة محليًا ويُستعاد تلقائيًا عند فتح التطبيق لاحقًا.
- تبسيط محتوى نافذة "حول" لتعرض لغة واحدة نشطة بدل عرض العربي والتركي معًا كما كان سابقًا، مع تعميم نص الوصف ليشمل أي خط سير (بدل تخصيصه بنزيب-عنتاب فقط).

## الإصدار 1.2 — 2026-07-10
- **زر تعديل مباشر لأسماء المدينتين**: أيقونة ✎ صغيرة بين اسمي المدينتين في بطاقة المسار تفتح نافذة تعديل اسم المدن مباشرة (بالإضافة إلى إمكانية التعديل من شريط خطوط السير العلوي).
- **خطوط سير متعددة محفوظة**: شريط قابل للتمرير أفقيًا أعلى الصفحة يعرض كل خطوط السير المحفوظة (مثال: "نزيب ⇄ عنتاب"، "عنتاب ⇄ كلس")، مع زر "+ خط سير جديد" لإضافة خط سير بمدينتين مخصّصتين عبر نافذة صغيرة.
- **تعديل أسماء المدن**: زر "✎" على كل خط سير يفتح نافذة لتغيير اسم المدينتين، أو حذف خط السير بالكامل (مع تأكيد ومسح بياناته المخزّنة، ولا يمكن حذف آخر خط سير متبقٍ).
- **عزل كامل للبيانات بين خطوط السير**: كل خط سير (وكل اتجاه فيه) يحتفظ بإشاراته وأزمنته وتسمياته الخاصة به بشكل مستقل تمامًا؛ عند التبديل بين خطوط السير أو العودة لاحقًا إلى خط سير سابق (مثل الرجوع من "عنتاب ⇄ كلس" إلى "نزيب ⇄ عنتاب")، تظهر كل بياناته كما تُركت تمامًا.
- **توافق تام مع البيانات القديمة**: خط السير الافتراضي (نزيب ⇄ عنتاب) يستخدم نفس مفاتيح التخزين السابقة، فلا تُفقد أي بيانات مسجَّلة من الإصدارات السابقة.
- الترويسة العلوية (NİZİP ⇄ GAZİANTEP سابقًا) وأزرار تبديل الاتجاه ونصوص المدينتين أصبحت كلها ديناميكية بالكامل حسب خط السير النشط.

## الإصدار 1.1 — 2026-07-10

  - **إعادة ترتيب** الإشارات بأزرار ▲ / ▼ لكل صف
- عدد الإشارات أصبح ديناميكيًا بالكامل (لم يعد ثابتًا على ١٠) — يُقرأ من طول مصفوفة البيانات المخزّنة لكل اتجاه، وتُحدَّث كل الشاشات (الترويسة، بطاقة الإشارة، الإحصائيات) تلقائيًا.

## ملاحظات للنشر على GitHub Pages
1. ارفع محتوى مجلد المستودع (`index.html`, `manifest.json`, `icons/`) كما هو إلى جذر الفرع (أو مجلد `/docs`).
2. فعّل GitHub Pages من إعدادات المستودع (Settings → Pages) واختر الفرع/المجلد المناسب.
3. الرابط الناتج سيفتح `index.html` تلقائيًا كصفحة رئيسية.
